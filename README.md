# ProDrive IQ 🚗

## Overview
ProDrive IQ is a web-based car service booking platform designed for users in Saudi Arabia.  
The platform allows users to find nearby workshops, compare service prices, and view booking options easily.

This project is developed as part of a Web Business Applications course assignment, currently, (Milestone 1) is focusing on UI/UX design and application planning.

---

## Problem Statement
Car owners often face difficulties when:
- Finding trusted workshops  
- Comparing service prices  
- Booking appointments efficiently  

There is no simple platform that combines all these features in one place.

---

## Proposed Solution
ProDrive IQ provides a simple and user-friendly solution by:
- Listing workshops across Saudi Arabia  
- Showing service prices clearly in SAR  
- Presenting a booking interface for service scheduling  

---

## Key Features (Milestone 1)
The current prototype includes the following features:

- Search interface for car services or workshops  
- Display of service providers with ratings and prices  
- Booking interface with date and time selection  
- User dashboard layout for viewing bookings and service history  
- Login and registration interface
